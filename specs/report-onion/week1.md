# 关键待讨论事项
- 健康日报图表形式探讨
  - 场景: 日报健康页面下的血压、心率等指标展示。
  - 问题: 用户的一天内测量数据具有不定时、非连续的特征。如果坚持使用标准的 24 小时轴折线图，会导致图表左右两侧出现大量空白区域，视觉效果松散且利用率低。
  - 当前方案: 已在前端实现算法，自动裁剪时间轴左右的无数据留白区域，强制聚焦于有数据的核心时间段。
  待决策:
  - 方案 A: 维持折线图 + 自动裁剪空白。优点是保留了“趋势”的连续感，能在视觉上暗示数据走向。
  - 方案 B: 改用柱状图。优点是天然适合展示离散的、不定时的数据点，不会因为时间间隔拉长而产生误导性的长连线。需要评估哪种形式更符合产品对“趋势回顾”的定义。
